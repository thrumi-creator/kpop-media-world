## 사업 범위

케이팝미디어월드는 다음과 같은 K-컬처 사업을 전개하고 있습니다:

- **K-Music**: 글로벌 시장을 겨냥한 K-POP 아티스트 발굴 및 육성, 음반 제작 및 유통
- **K-Content**: 다양한 엔터테인먼트 콘텐츠 기획 및 제작, 디지털 플랫폼 콘텐츠 개발
- **K-Beauty**: K-POP 스타와 연계한 뷰티 브랜드 협업 및 마케팅
- **K-Drama**: 드라마 및 영화 제작, 소속 아티스트의 연기 활동 지원

## 주요 사업

- **코리아아카데미**: 체계적인 트레이닝 시스템을 통한 차세대 K-POP 스타 양성
- **오디션**: 글로벌 오디션을 통한 신인 발굴 및 데뷔 지원
- **콘서트**: 국내외 공연 기획 및 제작, 팬미팅 및 이벤트 운영

## 웹사이트 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 반응형 디자인, 그라디언트, 애니메이션
- **JavaScript**: 인터랙티브 기능, 스크롤 효과, 이미지 갤러리
- **Google Fonts**: Noto Sans KR, Montserrat

## 주요 기능

1. **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기에서 최적화된 화면 제공
2. **스무스 스크롤**: 부드러운 페이지 내 네비게이션
3. **인터랙티브 갤러리**: 클릭하면 확대되는 이미지 갤러리
4. **애니메이션 효과**: 스크롤 시 페이드인 효과
5. **모바일 메뉴**: 햄버거 메뉴를 통한 모바일 네비게이션

## 사용 방법

1. 웹 브라우저에서 `index.html` 파일을 엽니다.
2. 또는 로컬 서버를 실행합니다:
   ```bash
   python3 -m http.server 8000
   ```
3. 브라우저에서 `http://localhost:8000`으로 접속합니다.
